Modalify

Examples

×

Basic


Any html element can be used as the target


$("#simple").click(function(){
  $("#simpleUsage").modalify();
});
×

Toggle option


Any jQuery Toggle animation is supported

$("#optionToggle").modalify({toggle:"slideToggle"});
×

Speed option


milliseconds of time, or jquery time string

$("#speed").click(function(){
  $("#optionSpeed").modalify({speed: 500});
});
×

Height 500px


$("#height").click(function(){
  $("#optionHeight").modalify("open", {height: "500px"});
});
