Modalify

Examples

Basic


Any html element can be used as the target


$("#simpleUsage").modalify();

Toggle option


Any jQuery Toggle animation is supported

$("#toggle").modalify({toggle:"slideToggle"});

Speed option


milliseconds of time, or jquery time string

$("#simple").modalify({speed: 500});

$("#simple").modalify({speed: "slow"});

Height 500px


$("#height").modalify({height: "500px"});
